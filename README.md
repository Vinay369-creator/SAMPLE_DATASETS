# CSV Dataset Collection for Django Practice

This repository contains multiple CSV datasets collected for practicing Django projects, database operations, data handling, filtering, searching, analytics, and REST APIs.
The datasets include small, medium, and large-scale data for real-world project development practice.

# Purpose of This Repository

These datasets are uploaded for:

- Django CRUD practice
- Database design practice
- ORM queries
- Search & filtering
- Pagination
- Dashboard development
- Chart visualization
- REST API development
- Authentication-based projects
- Admin panel practice
- Large dataset handling


# Technologies You Can Practice

- Django
- Django REST Framework
- SQLite
- PostgreSQL
- Pandas
- HTML/CSS/JavaScript
- Bootstrap
- Chart.js

# Important Note

Some datasets may contain large CSV files intended for performance and optimization practice.

Recommended topics to practice with large datasets:

- Bulk Insert

- Search Optimization

# Repository Goal
The main goal of this repository is to provide datasets for improving backend development and full-stack Django project skills using practical real-world data.
